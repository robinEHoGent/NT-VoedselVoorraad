# Fronted Application

## Pages & Functionality

### Home Page

Overview of the current cooking possibilities and recent activity

**User can:**

- See how many recipes are cookable with current inventory
- Go to almost expired products
- See which recipes are cookable and not cookable
- View a few favorite recipes
- View a few recently cooked recipes

---

### Recipe Book Page

Browse and add recipes

**User can:**

- View all recipes
- Filter recipes by tags
- Search recipes by name
- View recipe details and instructions

---

### Shopping List Page

Manage products that need to be purchased

**User can:**

- View all shopping list items
- Search items by name
- Add checked items to inventory

---

### Inventory Page

Manage available products and quantities

**User can:**

- View all products in inventory
- Search products by name
- Filter on expiration date
- Use products
- Change the storage place of a product
- See expired or almost expiring products

---
