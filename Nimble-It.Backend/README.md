# Backend API

## Tech Stack

- .NET
- Entity Framework
- REST API

---

## Running the backend

1. set environment variables MYSQL_CONNECTION_STRING,AZURITE_BLOB_CONNECTIONSTRING,Cors\_\_AllowedOrigins,Cors\_\_AllowedMethods
2. make a container with the image quintenst/pw2:backend
